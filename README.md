# Quiz Web Application

##  Overview
The Quiz Web Application is a simple, interactive web-based quiz system designed to help users test their knowledge through True/False questions. It is built using core front-end technologies and focuses on clean UI, input validation, and automatic score calculation.

This project is suitable for beginners and academic submissions, especially for demonstrating HTML, CSS, and JavaScript fundamentals.

---

##  Objectives
- To develop an interactive and user-friendly quiz application  
- To validate user inputs before submission  
- To calculate quiz scores dynamically using JavaScript  
- To demonstrate front-end development concepts and DOM manipulation

##  Features
- User information form (Name, Email, Date of Birth)
- 10 True/False quiz questions
- Mandatory input and answer validation
- Automatic score calculation
- Clean, professional, and responsive UI
- Works smoothly on all modern browsers

---

##  Application Flow
1. User opens the quiz webpage  
2. Enters personal details (Name, Email, DOB)  
3. Answers all 10 True/False questions  
4. Clicks the **Submit** button  
5. JavaScript validates inputs and responses  
6. Final score is displayed using an alert popup  

---

##  Scoring Mechanism
- Initial score starts at **0**
- Each correct answer increments the score by **1**
- Incorrect answers do not affect the score
- Final score is shown immediately after submission

##  Advantages

- Suitable for assignments, demos, and practice projects  

---

##  Limitations
- No database to store user results  
- No timer functionality  
- Questions are static (no randomization)  
- Supports only True/False questions
